## 🚀 快速开始

### 1️⃣ 安装（或者直接让openclaw安装）

```bash
# 克隆到 skills 目录
git clone https://github.com/xingzewu24/openclaw-dut.git \
  ~/.openclaw/workspace/skills/openclaw-dut

# 安装 Python 依赖
pip install requests beautifulsoup4 python-pptx pdfplumber handright Pillow reportlab pycryptodome chinesecalendar selenium
```

### 2️⃣ 配置

**一键配置（推荐）：**

```bash
cd   ~/.openclaw/workspace/skills/openclaw-dut
python3 scripts/setup.py    # Windows: python scripts/setup.py
```

交互式向导会逐步引导你完成所有配置，自动测试连接。也可手动编辑 `config.json`，详见下方认证指南。

**配置向导使用说明：**

- 每项服务开头会询问 `是否配置 XXX？(Y/n)`，回车进入配置，输入 `n` 跳过
- 配置项中回车保留 `config.json` 中的当前值
- 邮箱和超星登录均支持最多 5 次重试

## 🔑 认证配置详解

> 不是所有服务都必须配置，按需开启。

### 📋 超星学习通

超星学习通是核心功能的基础，**必须配置**。

**Step 1：** 确认你在学习通上有账号

**Step 2：** 确认你的超星登录手机号和密码

**Step 3：** 在 `config.json` 中填入配置：

```json
{
  "chaoxing_phone": "你的手机号",
  "chaoxing_password": "你的超星密码"
}
```

**验证是否成功：** 运行 `python3 scripts/chaoxing_api.py courses` 看到课程列表就说明配置成功 ✅

> ⚠️ 密码会以明文存储在 config.json 中（已在 .gitignore 中排除），请勿分享该文件。

---

### 📧 大工邮箱

用于查看未读邮件、搜索邮件、发送邮件。通过 IMAP/SMTP 协议直连 `mail.dlut.edu.cn`。

**Step 1：** 确认你的 DUT 统一身份认证账号

**Step 2：** 在 `config.json` 中填入配置：

```json
{
  "mail_username": "你的邮箱前缀",
  "mail_password": "你的密码",
  "dlut_mail_domain": "你的邮箱后缀"
}
```

> ⚠️ **注意邮箱域名：** 大工邮箱有两种域名后缀（`mail.dlut.edu.cn` 和 `dlut.edu.cn`），请在 `dlut_mail_domain` 中填写你实际使用的域名。配置向导会引导你选择。

**验证是否成功：** 对 AI 说「有没有新邮件」，看到邮件列表就说明配置成功 ✅

> 💡 **安全建议：** 如果学校支持应用专用密码，建议使用应用专用密码而非主密码。
>
> ⚠️ `config.json` 已在 `.gitignore` 中，不会被提交到 Git。但请确保不要手动上传。

### 🎓 教务系统 / 门户系统 / 校园网自助服务

这三套系统共用同一套 DUT 统一身份认证（CAS SSO）凭证：

- **教务系统** (`jxgl.dlut.edu.cn`)：课表、考试、成绩
- **门户系统** (`portal.dlut.edu.cn`)：统一身份认证门户、应用集成
- **校园网自助服务** (`tulip.dlut.edu.cn`)：网费余额、流量查询

**Step 1：** 确认你的 DUT 统一身份认证账号

**Step 2：** 在 `config.json` 中填入配置：

```json
{
  "dlut_username": "你的学号",
  "dlut_password": "你的密码"
}
```

**验证是否成功：**
- 教务：`python3 scripts/dlut_jxgl.py login` ✅
- 门户：`python3 scripts/dlut_portal.py login` ✅
- tulip：`python3 scripts/dlut_tulip.py login` ✅

## 📖 能干什么？直接跟 AI 聊天就行

> 装好之后**不需要记任何命令**，直接用自然语言对话。以下是最受欢迎的使用场景：

### 🔥 作业辅导 — 最核心的功能

| 你说                             | AI 帮你做                                             |
| -------------------------------- | ----------------------------------------------------- |
| "我有什么作业没交？"             | 拉取所有未交作业，**按截止时间分三类**：未截止待交（含未开始） / 已截止未交（带过期天数） / 无截止时间 |

| "帮我把 DDL 同步到日历"          | 导出 ICS，一键导入 Apple / Google / Windows 日历      |

> 💡 **工作流示例：** 你说「这周热力学作业帮我辅导一下」→ AI 自动拉取作业要求 → 下载课件 → 提取关键公式和知识点 → 逐题给出解题思路和参考答案框架。

## 📸 实战效果

> 以下为真实对话场景（已脱敏），展示 AI 助手的实际使用效果。

### 🔥 "帮我看看这周有什么作业，辅导一下"

**第一步：AI 自动拉取作业列表（按截止时间分三类）**

```
📋 未交作业总览（共 6 个）

⏳ 未截止待交（4 个）
  🟡 [数据结构] 作业8 → 2026-04-28 23:59 (剩 49h) ⚡紧急
  🟢 [大学物理] 物理作业七 → 2026-05-02 23:59 (剩 145h)
  🟢 [工程热力学] 第二次作业 → 2026-05-08 23:59 (剩 289h)
  🆕 [毛概] 实践作业 → 未开始（2026-05-10 08:00 起）

❌ 已截止未交（1 个）
  🔴 [线性代数] 作业5 → 2026-04-20 23:59（已过期 5 天）

❓ 无截止时间（1 个）
  ⚪ [思政] 课堂讨论
```

**第二步：AI 下载课件 + 读取作业要求 → 逐题辅导**

```
📝 数据结构 作业8 辅导

题目 1: 二叉搜索树的插入与删除
  📖 参考课件: 第7讲-二叉搜索树.pdf 第12-18页
  💡 思路: 插入操作递归比较 key 值，找到空位插入；
     删除分三种情况：叶子节点直接删、单子节点替换、
     双子节点找中序后继替换后递归删除...
  ✅ 关键公式: 平均查找长度 ASL = (1/n)Σ(level_i)

题目 2: AVL 树旋转
  📖 参考课件: 第8讲-平衡二叉树.pdf 第5-11页
  💡 思路: 判断失衡类型 (LL/RR/LR/RL)，
     LL → 右旋，RR → 左旋，LR → 先左旋后右旋...
```
